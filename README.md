# Disease Risk Predictor
Predicts diabetes risk from patient vitals using ML.NET FastTree 
classifier trained on the Pima Indians dataset.

**Tech:** ML.NET · ASP.NET MVC · C# · Bootstrap 5

## Model Performance
| Metric | Score |
|--------|-------|
| Accuracy | 78.5% |
| F1 Score | 0.76 |
| AUC-ROC | 0.83 |

## How to run locally
````bash
git clone https://github.com/yourname/DiseaseRiskPredictor
cd DiseaseRiskPredictor
dotnet run
````
